# Tooltip

This directive adds a tooltip to the element it is used on.

## Simple usage

```html
<template>
  <div>
    <i
        class="fa fa-question-circle float--right u-mt-0_125"
        v-tooltip="`some text to display in a tooltip`" />
  </div>
</template>
<script>
import { Tooltip } from 'demosplan-ui/directives'

export default {
  name: 'MyComponent',

  directives: {
    tooltip: Tooltip
  },

  //...
}

</script>
```
